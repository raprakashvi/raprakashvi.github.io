# Paper PDFs

Place PDF files here and link them from your publications.

## Usage

In `index.html`, link to a local PDF:

```html
<a href="papers/tumormap.pdf" class="pls" target="_blank">PDF</a>
```

Or use external URLs (arXiv, journal pages):

```html
<a href="https://arxiv.org/pdf/2511.05723" class="pls" target="_blank">PDF</a>
```

## Naming

Use short, descriptive names: `tumormap.pdf`, `see-plan-cut.pdf`, etc.
