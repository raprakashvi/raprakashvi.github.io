# Publication Images

Place one image per paper here. Used as thumbnails in the Publications section.

## Usage

In `index.html`, inside each `.pi-thumb` div, replace the placeholder with:

```html
<img src="images/publications/tumormap.jpg" alt="TumorMap">
```

## Supported formats

- `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
- Recommended size: 120×80px or similar aspect ratio (3:2)
- For video thumbnails: use `<video src="images/publications/demo.mp4" muted autoplay loop playsinline></video>`

## Naming

Match the paper: `tumormap.jpg`, `see-plan-cut.jpg`, `palpaid.gif`, etc.
